HTML

  • the head element contains infomration about the webpage
  • The body element represents the visible content shown to the user
  • The h1 is the opening tag and /h1 is the closing tag.
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • the a element (which stands for anchor) creates links to the same webpage or other webpages.
  • The a href attribute points to the URL for the link

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add [file] add
  • git clone ssh://user@domain.com/repo.git:Clone and Existing Repository
  • git add .:Add all current changes to the next commit
  • git add -:Add some changes in to the next commit
  • git commit -a:Commit all local changes in tracked files
  • git add -p:Commit previously staged changes
  • git branch:List all existing branches
  • git commit:Add some changes in to the next commit
  • git checkout:Switch HEAD branch
  • git branch:Create a new branch based on your current HEAD
  • git checkout --track:Create a new tracking branch based on a remote branch
  • git merge:Merge branch into your current HEAD
  • git reset --hard:Discard all local changes in your working directory
